# TimeXer 终极优化训练命令

本文件基于对 `summary_metrics.csv` 的深度分析以及对数据集结构的重新梳理，为您制定了针对 .300BLK 和 5.56 系列子弹的**终极优化策略**。

我们的目标是“不惜一切代价提高精度”，因此我们打破了单一的数据集限制，为每种子弹选择了其历史上表现最好的**数据源**（Solo vs Category）和**特征模式**（S vs MS vs M）。

### 策略核心

1.  **5.56 系列 (M855, M855A1, M995)**：
    *   **最佳方案**：**集合训练 (Collection Category)**
    *   **原因**：历史数据显示，5.56 系列的子弹之间存在显著的相关性，且外生变量（Collection）有帮助。Category 模型的整体误差 (MSE ~0.21) 显著优于大多数 Solo 模型 (M855 单独训练 MSE 高达 0.52)。
    *   **配置**：`Seq 192` (最稳健长度) + `Dropout 0.3` (增强正则化) + `M` 特征。

2.  **.300BLK 3**：
    *   **最佳方案**：**独立训练 (Solo MS)**
    *   **原因**：这是唯一一个依赖长序列 (Seq 336) 和大模型 (Large) 的任务。Category 模型的短序列设置无法满足它的需求。
    *   **配置**：`Seq 336` + `Large` + `Dropout 0.3`。

3.  **.300BLK 5**：
    *   **最佳方案**：**独立纯净训练 (Solo S)**
    *   **原因**：这是唯一一个**外生变量起反作用**的任务。使用不带 collection 的纯净数据 (Solo S) 取得了全场最佳 MSE (0.09)。
    *   **配置**：`Seq 96` + `S` 特征 + `dataset/bullet/solo` (非 collection)。

4.  **.300BLK 4**：
    *   **最佳方案**：**独立训练 (Solo MS)**
    *   **原因**：在 Seq 96 下表现优异。
    *   **配置**：`Seq 96` + `MS` 特征 + `Dropout 0.3`。
